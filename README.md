# Instagram Manual Testing Project

## Overview
Manual test cases written for the login functionality of instagram.com
as part of a personal QA portfolio.

## Scope
- Functional testing (valid/invalid login flows)
- Edge case testing (special chars, max length, case sensitivity)
- Usability testing (forgot password, remember me)

## Tools Used
- Browser: Chrome
- Documentation: Markdown / GitHub
- Bug tracking: GitHub Issues
